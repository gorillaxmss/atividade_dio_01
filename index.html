Introdução

Atualmente, a internet faz parte do nosso dia a dia, e por trás de todos os sites que acessamos existe uma linguagem chamada HTML. HTML significa HyperText Markup Language e é usada para estruturar o conteúdo das páginas da web, como textos, títulos, listas e links.

Neste trabalho, vou explicar algumas tags básicas do HTML, mostrando para que servem e dando exemplos simples para facilitar o entendimento.

Títulos no HTML (h1 até h6)

As tags "< h1> até < h6>" são usadas para criar títulos e subtítulos em uma página. O < h1> é o título principal, enquanto o < h6> é o menor e menos importante. Essas tags ajudam a organizar o conteúdo e facilitam a leitura.

Exemplos:

h1 Título Principal

h2 Subtítulo

h3 Outro título

Parágrafo (< p>)

A tag < p> serve para escrever parágrafos de texto, como este que estamos lendo agora. Ela organiza o conteúdo e deixa a página mais fácil de entender.

Formatação de Texto

O HTML possui várias tags que ajudam a destacar partes do texto:

  • < strong> deixa o texto em negrito, mostrando importância
  • < i> deixa o texto em itálico
  • < u> sublinha o texto
  • < mark> destaca o texto
  • < small> diminui o tamanho do texto
  • < del> risca o texto, indicando algo removido

Exemplo:
Texto importante,
em itálico,
sublinhado,
destacado,
menor
e removido.

A tag < font>

A tag era usada antigamente para mudar a cor e o tamanho do texto.
Hoje em dia, ela não é mais recomendada, mas ainda aparece em alguns códigos antigos ou em atividades escolares.
Exemplo: Texto azul

Listas (< ol>, < ul> e < li>)

As listas servem para organizar informações:

  • < ol> cria listas numeradas
  • < ul> cria listas com marcadores
  • < li> representa cada item da lista

Lista Ordenada

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada

  • Item A
  • Item B
  • Item C

Links (< a>)

A tag permite criar links, que levam o usuário para outros sites ou páginas.
Exemplo:

Site para aprender tudo sobre tecnologia

Linha Horizontal (< hr>)

A tag < hr> cria uma linha que serve para separar partes do conteúdo, deixando a página mais organizada.

Subscrito e Sobrescrito (< sub> e < sup>)

Essas tags são muito usadas em fórmulas:

  • < sub> coloca o texto para baixo
  • < sup> coloca o texto para cima

Fórmula da água: H2O

Potência matemática: 23 = 8

Citação (< blockquote>)

A tag < blockquote> é usada quando queremos colocar uma citação maior dentro do texto.

"A tecnologia move o mundo." – Steve Jobs

Abreviação (< abbr>)

A tag < abbr> serve para explicar siglas. Quando o mouse passa por cima, aparece o significado.
